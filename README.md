# AI Content Factory for Mascot Videos

Автоматизированная фабрика генерации видео с маскотами на базе n8n + HEYGEN.

## Проблема
Ручное создание и монтаж промо-видео с маскотами занимало дни и требовало команды редакторов. Масштабировать производство было невозможно.

## Решение
- n8n workflow: от брифа → генерация скрипта (LLM) → видео (HEYGEN/RunWay) → монтаж (Submagic / WESHOP) → публикация в Telegram-бот
- Персонализация через RAG-подобные элементы (база шаблонов маскотов)
- Multi-agent подход: отдельные агенты для текста, видео, монтажа
- Self-service интерфейс в Telegram для клиентов

## Ключевые результаты
- Стабильная генерация 3–10 видео в день без ручного вмешательства
- Сокращение времени производства на 70–80%
- Масштабирование контента для маркетинга/рекламы без роста штата

## Стек
- n8n
- HEYGEN, Submagic
- LLM (Claude / Grok / OpenAI) для скриптов и промптов
- Telegram Bot API
